# Submission Checklist (Aligned to Brief)

## Before Minerva Submission
- Public GitHub repository link included in report.
- Repository has clear commit history (not one bulk commit).
- `README.md` includes setup/run instructions.
- API documentation PDF exists and is linked in README.
- Technical report includes GenAI declaration and logs appendix.
- Slides include: version control, API docs, report highlights, deliverables.
- Demo code version exactly matches submitted repository.

## Dry-Run Demo Checks
- `npm install`
- `npm run build`
- `npm test`
- `npm run dev`
- Verify key endpoints in Postman/Swagger:
  - auth flow
  - ingredient CRUD
  - recipe CRUD
  - analytics outputs

## Academic Integrity Checks
- Dataset and library citations included.
- GenAI usage transparently documented.
- Any borrowed code patterns clearly attributed.
